Troubleshooting

  • Camera not working: Make sure you're using HTTPS or localhost, and allow camera permissions
  • QR not detected: Ensure good lighting and hold the QR code steady
  • Backend errors: Make sure the backend server is running on port 5000
  • Authentication errors: Make sure you're logged in as an admin user
